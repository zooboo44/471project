LANGUAGE:
The FTP server and client is written in Python. Python3 will be needed to run the program.


USAGE:
Run the server `python3 server.py`
Run the client `python3 client.py`
Client commands:
list
`list` will list the files on the server.
get
`get <filename>` will download the file from the server to the client
put
`put <filename>` will upload the file from the client to the server
quit
`quit` will quit the client
